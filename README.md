## :dart: About ##

AI based django practice codes with various tools like Stripe, Twilio etc.

## :sparkles: Features ##

:heavy_check_mark: Stripe Payments;\
:heavy_check_mark: Twilio API;\
:heavy_check_mark: Webhook;\
:heavy_check_mark: Websocket;

## :rocket: Technologies ##

The following tools were used in this project:

- [Django](https://www.djangoproject.com/)

## :white_check_mark: Requirements ##

Before starting :checkered_flag:, you need to have [Git](https://git-scm.com) and [Django](https://www.djangoproject.com/) installed.

## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/Maruf346/ai-django-prac

# Access
$ cd ai-django-prac

# Install dependencies
$ pip install -r requirements.txt

# Run the project
$ py manage.py runserver

# The server will initialize in the <http://localhost:8000>
```
